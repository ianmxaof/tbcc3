Model search

Select a username on any page → right‑click → TBCC — Look up username → choose one source or Search all enabled sites. Built‑in sites ship with the extension; add your own below (URL must include {username}).

Custom search URL

Paste the URL from the address bar after you run a manual search on that site (so you get the real query format). Put {username} where the search term goes — often ?q={username} or &query={username}, not a guessed path like /artists/name (many sites 404 on those).

Sources

Toggle sources on or off. Only enabled sources appear in the context menu.

Built‑in list: extension/model-search-sites.json. Custom sources are stored in this browser. Reload the extension after editing the JSON file.

Reverse image search

Right-click an image → Reverse image search (fan-out). Engines take a public image URL (http/https). Edit extension/reverse-image-sites.json to add engines.
