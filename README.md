# telegram_bot2

Monorepo containing **TBCC** (Telegram Bot Command Center): FastAPI backend, React dashboard, Chrome extension, Celery workers, and Telegram bots for media workflows, scheduling, subscriptions, and channel posting.

**Documentation, setup, and operations:** see **[tbcc/README.md](tbcc/README.md)**.

**Remote:** typically `origin` → GitHub (e.g. `tbcc3`). On clone, `cd tbcc` and follow Quick start in `tbcc/README.md`.
